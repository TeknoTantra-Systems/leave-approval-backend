# Leave Approval Backend

Employee Leave Approval Workflow application built with Spring Boot and Camunda.

## Tech Stack
- Java 17
- Spring Boot 3.2
- Camunda Community Edition 7.20
- PostgreSQL
- Maven

## Getting Started
```bash
docker-compose up -d
```

## API Documentation
http://localhost:8080/swagger-ui.html
